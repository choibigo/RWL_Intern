- 도커 이미지 삭제가 불가능한 경우
    - 컨테이너에서 사용되고 있는 이미지는 삭제할 수 없다.
    
    ![image.png](image%202.png)
    
- 컨테이너에서 사용하고 있는 이미지이더라도 강제로 삭제가 가능
    - 중단된 컨테이너에서 사용하고 있는 이미지는 삭제를 해주는 것이 맞다. 하지만 실행 중인 컨테이너에서 사용되고 있는 이미지는 삭제해주지 못한다
    
    ![image.png](image%203.png)
    
    ![image.png](image%204.png)
    
- 컨테이너에 사용하고 있지 않은 전체 이미지들을 삭제하는 명령어
    
    ```python
    docker image rm $(docker images -q)
    ```
